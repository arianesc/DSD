# DSD
Desenvolvimento de Sistemas Distribuídos
Esse é um exemplo RMI de armazenamento de dados.

1. Criar ambiente virtual
```
python -m venv env
```

2. Ativar ambiente virtual
```
source env/bin/activate
```
3. Instalar rpyc
```
pip install rpyc
```
4. Rodar Servidor:
  ```
python servidor.py
```

5. Rodar Cliente:
```
python cliente.py
```
